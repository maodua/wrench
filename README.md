## 安装

Gradle:
```groovy
dependencies {
    compile "io.github.maodua:模块名称:版本号"
}
```
Maven:
```xml
<dependency>
    <groupId>io.github.maodua</groupId>
    <artifactId>模块名称</artifactId>
    <version>版本号</version>
</dependency>
```
